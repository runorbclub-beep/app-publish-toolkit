# 贡献指南

欢迎为 App Publish Toolkit 贡献内容！

## 如何贡献

### 1. 报告问题
- 在 [Issues](https://github.com/runorbclub/app-publish-toolkit/issues) 中报告问题
- 请提供详细的信息：平台、错误信息、复现步骤

### 2. 提交内容
我们欢迎以下类型的贡献：

- **新的审核被拒案例**：在 `docs/rejection-reasons.md` 中添加
- **平台政策更新**：更新相关文档
- **新的脚本工具**：在 `scripts/` 目录中添加
- **新的模板**：在 `templates/` 目录中添加
- **翻译**：帮助翻译文档
- **案例**：在 `examples/` 目录中添加你的上架案例

### 3. 提交 PR
1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交修改：`git commit -m 'Add: your feature'`
4. 推送：`git push origin feature/your-feature`
5. 创建 Pull Request

### 4. 内容规范
- 使用 Markdown 格式
- 中英双语优先
- 包含实战经验，不只是理论
- 标注信息来源和日期
