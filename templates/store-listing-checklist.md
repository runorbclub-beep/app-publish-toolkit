# App 上架清单 — 逐项检查

> 提交审核前逐项检查，避免被拒

## Apple App Store

### 账号与证书
- [ ] Apple Developer 账号已激活（$99/年）
- [ ] Distribution Certificate 已创建并安装
- [ ] App ID 已注册（Bundle Identifier）
- [ ] Provisioning Profile（App Store）已创建并安装
- [ ] App Store Connect API Key 已创建（如使用 Codemagic）

### App 配置
- [ ] Bundle Identifier 正确
- [ ] Version 和 Build 号正确
- [ ] 应用图标 1024×1024 已配置
- [ ] Info.plist 中所有权限说明完整（NSXxxUsageDescription）
- [ ] 支持的设备方向正确
- [ ] 最低 iOS 版本设置合理

### App Store Connect
- [ ] App 已创建
- [ ] App 名称（30字符内）
- [ ] 副标题（30字符内）
- [ ] 描述（4000字符内）
- [ ] 关键词（100字符内，逗号分隔）
- [ ] 推广文本（170字符内）
- [ ] App 分类正确
- [ ] 隐私政策 URL 可直接访问（无重定向！）
- [ ] 支持 URL 可访问
- [ ] 营销 URL 可访问（可选）
- [ ] 版权信息
- [ ] App 图标 1024×1024 PNG
- [ ] 截图：6.7寸（1290×2796）至少5张
- [ ] 截图：6.5寸（1242×2688）至少5张
- [ ] App 预览视频（可选）

### 审核信息
- [ ] 审核联系人信息
- [ ] 演示账号（如需登录）
- [ ] 审核备注（特殊说明）

### 构建与提交
- [ ] IPA 已构建并上传
- [ ] 构建版本已选择
- [ ] 出口合规信息已填写
- [ ] 提交审核

- [ ] 类别正确

## 通用检查

### App 质量
- [ ] 无崩溃、无 ANR
- [ ] 所有功能可用
- [ ] 无占位内容
- [ ] 无 placeholder 文本
- [ ] 适配不同屏幕尺寸
- [ ] 支持最新 OS 版本
- [ ] 离线功能正常（如适用）

### 内容合规
- [ ] 无侵权内容
- [ ] 无不当内容
- [ ] 无误导信息
- [ ] 权限使用合理
- [ ] 第三方 SDK 已声明

### 隐私政策检查
- [ ] URL 可公开访问
- [ ] 无重定向（直接返回 HTML）
- [ ] 内容与数据安全表一致
- [ ] 包含所有必要条款
- [ ] 联系方式有效
