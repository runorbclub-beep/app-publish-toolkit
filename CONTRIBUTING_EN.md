# Contributing Guide

Welcome to contribute to App Publish Toolkit!

## How to Contribute

### 1. Report Issues
- Report problems in [Issues](https://github.com/runorbclub/app-publish-toolkit/issues)
- Please provide detailed information: platform, error message, reproduction steps

### 2. Submit Content
We welcome the following types of contributions:

- **New rejection cases**: Add to `docs/rejection-reasons.md` / `docs/rejection-reasons-en.md`
- **Platform policy updates**: Update relevant docs
- **New script tools**: Add to `scripts/` directory
- **New templates**: Add to `templates/` directory
- **Translations**: Help translate docs
- **Case studies**: Add your publishing experience to `examples/` directory

### 3. Submit PR
1. Fork this repository
2. Create branch: `git checkout -b feature/your-feature`
3. Commit changes: `git commit -m 'Add: your feature'`
4. Push: `git push origin feature/your-feature`
5. Create Pull Request

### 4. Content Guidelines
- Use Markdown format
- Bilingual (Chinese + English) preferred
- Include real-world experience, not just theory
- Mark information source and date
